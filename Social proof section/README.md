# Social Proof Section

A responsive social proof landing page built with HTML and CSS for the ASTUMSJ Bootcamp project.

## Project Overview

This project recreates a polished social proof section featuring:
- a strong hero message
- customer rating highlights
- testimonial cards for verified buyers
- a responsive layout for desktop and mobile screens

## Features

- Responsive design
- Clean semantic HTML structure
- Modern CSS styling
- Ready-to-view static webpage

## Project Files

- index.html — main page content
- style.css — visual styling and layout
- images/ — icons and profile images
- design/ — design reference files

Open index.html in your browser to view the project locally.
